# Pet Gram

## 概要
#### Petに特化したSNSサービス

## 機能一覧
- 記事投稿機能
- 記事削除機能
- 記事編集機能
- 投稿一覧表示
- 投稿別詳細機能
- いいね（Ajax）
- ユーザーがいいねした記事別機能
- 投稿タグ機能
- 投稿タグ別一覧
- コメント機能
- 画像ファイルアップロード機能
- フォロー・フォロワー機能
- ユーザー登録
- ユーザーログイン機能
- DBテーブルリレーション機能

## 技術一覧
- 言語　- PHP
- 開発環境　-MAMP
- フレームワーク/ライブラリ - Larabel / BootStrap / Vue.js / JQuery
- DB - ローカル/MySQL デプロイ後/PostgreSQL
- 画像のアップロード　-DBにbase64をいれ表示
- デプロイ　-heroku
